You will probably need to deal with Matlab's oddities in order to actually import a package.

>> import py.pyNastran.op2.op2.OP2.*
>> clear
>> import py.pyNastran.op2.op2.OP2.*
>> clear import
>> import py.pyNastran.op2.op2.OP2

Also, you will probably need to redirect standard out (see the subfolder).
